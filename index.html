What you get from one command

Feed it a pcap. Get everything you need.

Protocol Grammar

Human-readable rules describing the exact structure of your protocol. Magic bytes, type fields, length fields, variable data — all identified automatically.

Anomaly Detection

Every packet that deviates from the learned structure is flagged. Corrupted headers, unknown types, truncated messages. No signatures needed.

Wireshark Dissector

Auto-generated .lua dissector, registered on the right port. Load it in Wireshark — fields are named, anomalies are marked in Expert Info.

Fuzz Corpus

500 mutated packets in 3 strategies: field corruption, boundary values, structural mutations. Ready to send to the target.

Tested on real protocols

Binary and text, auto-detected.

MODBUS/TCP

DNS

Transaction ID, Flags, Questions count, Authority. Structural anomalies found in real captures.

Telnet

Auto-split: IAC binary commands separated from text data in the same flow.

HTTP / FTP

Text mode: tokens discovered (\r\n, HTTP, GET), requests and responses separated.
